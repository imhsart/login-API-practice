# 🔐 Signup / Login / Logout API Practice

This is a small practice setup to understand how API calling works in a real flow — especially for authentication.

It includes working with signup, login, logout APIs, and a simple dashboard feature using GET, POST, and DELETE requests.

---

## 🚀 What this includes

- User **Signup**
- User **Login**
- User **Logout**
- Dashboard with **random message generation**
- API calls using different methods:
  - GET
  - POST
  - DELETE

---

## 🧠 What I learned

- How authentication flow works (token-based)
- Handling APIs in frontend using JavaScript
- Storing and using tokens (for login/logout)
- Restricting actions for logged-in users only
- Basic request handling and response usage
